# 2019-intern-challenge
2019 intern interview refactoring and bug fixing challenge.

# Python

## Python Setup


#### Install python3

`$ brew install python3`


#### Setup virtual environment
`$ python3 -m venv .`

`$ source bin/activate`

## Python Testing 

`$ cd python`

`$ nosetests`

# JS

## JS Setup

#### Install NVM
`$ brew install nvm`

#### Install Node
`$ nvm install node`

#### Install Project Dependencies
`$ npm install`

## JS Testing
`$ npm test`
